# VeloxStack Marketing Site

Static marketing site for VeloxStack Mail with the following pages:

- `/` homepage
- `/pricing`
- `/docs`
- `/demo`

## Local structure

- `index.html`
- `pricing.html`
- `docs.html`
- `demo.html`
- `styles.css`

## GitHub Pages

This site is plain static HTML/CSS and can be published directly with GitHub Pages from the repository root.

Recommended Pages settings:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

After pushing to GitHub, enable Pages in the repository settings and GitHub will serve `index.html` automatically.
